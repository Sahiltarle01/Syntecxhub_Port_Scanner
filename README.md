# Syntecxhub Cybersecurity Internship Task 1

## Project: Port Scanner

This project is a Python-based TCP Port Scanner developed as part of the Syntecxhub Cybersecurity Internship.

## Features
- Scan open ports on a target host
- Multi-threaded scanning
- Fast and efficient scanning
- Service banner detection
- Error handling

## Technologies Used
- Python
- Socket Programming
- Threading

## How to Run

1. Install Python
2. Run the program

python port_scanner.py

3. Enter target domain or IP
4. Enter port range

## Example Target

scanme.nmap.org
